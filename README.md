<h1 align ="center"> 🎓 UniGrade — Student Grade Predictor</h1>

UniGrade is a Machine Learning web application that predicts student final grades using a Random Forest algorithm trained on student performance data. Built with React frontend 
and Python Flask backend. 

---

## 🌟 Features

- 🤖 Machine Learning model trained on student data
- 📊 Predicts final grade based on study habits
- 🎯 Grade letter and personalized feedback
- 📈 Grade scale reference chart
- 🌲 Random Forest algorithm for high accuracy

---

##  Tech Stack

| Part        | Technology              |
|-------------|-------------------------|
| Frontend    | React                   |
| Backend     | Python + Flask          |
| ML Model    | Scikit-learn            |
| Algorithm   | Random Forest Regressor |
| Data        | CSV Dataset             |

---

## 🧠 How the ML Model Works
```
50 students data loaded
        ↓
80% used for training
        ↓
20% used for testing
        ↓
Random Forest learns patterns
        ↓
Model saved as grade_model.pkl
```
